재밌는 건 바로 이 부분인데요.

한쪽에서는 차트 패턴을 분석해 기술적 분석 결과를 알기 쉽게 정리해주고요.

다른 쪽에서는 재무제표를 검토해 매출, 영업 이익 같은 수치를 토대로 한 기업 분석을 보여줍니다.

그리고 해당 기업에 대한 최신 뉴스 중에서 중요한 것만 골라서 정리해주고 있습니다.

마치 전문 애널리스트 팀이 종합 분석 보고서를 작성해주는 것처럼요!

"와... 멋져 보이는데, 이걸 정말 만들 수 있을까요?"

물론입니다! 여러분이 이미 알고 계신 파이썬과 ChatGPT API 지식을 기반으로,

필요한 도구들을 하나씩 더해가기만 하면 충분히 가능해요.

어떤 것들이 필요한지 하나씩 살펴볼까요?

먼저, 이런 시스템을 만들려면 실제 주가 데이터가 필요하겠죠?

증권사 API 연동을 통해 실제 시장 데이터를 받아오는 방법을 배워볼 거예요.

더 이상 수동으로 주가를 검색할 필요 없이, 코드 몇 줄로 원하는 데이터를 자동으로 수집할 수 있죠.

데이터 분석 시간에 배웠던 pandas(판다스)를 사용하면 주가 데이터도 편하게 다룰 수 있어요.

그리고 차트를 그리기 위해, plotly(플랏리)라는 라이브러리도 사용해보겠습니다.

자, 이제 기본적인 데이터와 차트는 준비가 됐네요. 그런데 아까 보셨던 AI 분석은 어떻게 구현할 수 있을까요?

여러분이 이미 익숙한 ChatGPT API만으로 가능합니다!

인공지능이 분석할 수 있게, 잘 정리된 프롬프트와 데이터를 전달하기만 하면 됩니다.

예전에는 차트 분석, 재무제표 분석, 뉴스 기사 정리 같은 기능은 프로그램으로 만드는 게 정말 어려웠어요.

거의 불가능하다고 해도 됐을 정도로요.

하지만 이제는 세상이 달라졌죠.

파이썬과 ChatGPT를 활용하면 얼마나 강력한 힘을 발휘할 수 있는지, 계속해서 느껴보세요!

자, 이제 실시간 데이터도 있고, 멋진 차트도 있고, AI의 분석 결과도 있네요.

하지만 이 모든 정보가 흩어져 있다면 불편하겠죠?

매번 여러 프로그램을 왔다갔다하면서 정보를 확인해야 한다면,

결정적인 순간에 중요한 정보를 놓칠 수도 있을 거예요.

그래서 최종적으로 Streamlit(스트림 릿)을 활용해 모든 정보를 한눈에 볼 수 있는 대시보드를 만들어볼 겁니다.

처음에 봤던 바로 그 화면을 바로 이 Streamlit으로 만들어 보는 거예요.

이렇게 통합된 대시보드는 여러분의 스마트한 투자 의사결정을 도와주는 든든한 투자 관제 센터가 될 겁니다!

그럼, 다음 시간부터 구체적인 방법들을 하나씩 자세히 살펴보도록 하겠습니다!

### 1️⃣ 주가 데이터와 차트

- 증권사 API를 통해 시장 데이터를 불러옵니다.
- 파이썬과 pandas를 활용해 데이터를 자동으로 수집하고 정리합니다.
- plotly 라이브러리로 주식 차트를 그립니다.

### 2️⃣ AI 주식 분석

- ChatGPT API를 사용해서 주식 정보를 분석합니다.
- 정리된 차트, 재무제표, 뉴스 기사 데이터와 프롬프트를 ChatGPT에게 전달합니다.
- 기술적 분석, 기본적 분석, 최신 뉴스가 있는 종목 분석 보고서를 완성합니다.

### 3️⃣ 대시보드 구축

- Streamlit을 사용하면 파이썬에서 웹 애플리케이션 제작과 데이터 시각화를 쉽게 할 수 있습니다.
- Streamlit을 활용해 주식 차트와 분석 보고서를 한 곳에 모아 둔 대시보드를 제작합니다.
